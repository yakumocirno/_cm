# 二次方程式根求解（root2）

## 問題說明

給定一個二次方程式：

[ f(x) = ax^2 + bx + c ]

請撰寫一個 Python 函數 `root2(a, b, c)`，利用**根的公式**計算並回傳該方程式的兩個解（根）。

根的公式為：

[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} ]

其中 ( b^2 - 4ac ) 稱為**判別式**。

---

## 解題思路

1. **計算判別式**
   判別式 ( \Delta = b^2 - 4ac ) 用來判斷根的型態：

   * ( \Delta > 0 )：兩個相異實數根
   * ( \Delta = 0 )：一個重根
   * ( \Delta < 0 )：兩個共軛複數根

2. **使用 `cmath` 模組**
   為了同時支援「實數根」與「複數根」，本程式使用 Python 的 `cmath` 模組，而不是 `math` 模組。`cmath.sqrt()` 可以正確處理負數開根號的情況。

3. **套用根的公式**
   根據公式計算兩個解：

   * ( x_1 = \frac{-b + \sqrt{\Delta}}{2a} )
   * ( x_2 = \frac{-b - \sqrt{\Delta}}{2a} )

4. **驗證結果正確性**
   撰寫輔助函數 `f(a, b, c, x)`，將算出的根代回原方程式，並使用 `cmath.isclose()` 檢查結果是否近似於 0。

---

## 程式說明

### `root2(a, b, c)`

* **輸入**：二次方程式係數 `a`, `b`, `c`
* **輸出**：一個 tuple `(x1, x2)`，代表兩個根（可能為複數）

### `f(a, b, c, x)`

* 用來計算 ( f(x) = ax^2 + bx + c )
* 主要用途是驗證根是否正確

---

## 測試案例

測試係數：

```python
a = 1
b = 2
c = 5
```

此時判別式：
[ \Delta = 2^2 - 4(1)(5) = -16 < 0 ]

因此應得到兩個複數根。

程式輸出結果顯示：

* 成功算出兩個複數根
* 將根代回原方程式後，結果近似於 0

代表解法正確。

本程式使用數學公式直接求解二次方程式，並透過 `cmath` 模組確保在所有情況（包含複數根）下都能正常運作。透過代回驗證，也能確認計算結果的正確性。
