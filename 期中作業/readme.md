# 以數值方法求解高次多項式方程式之研究與實作

## 專題簡介
本專題探討「當多項式次數大於等於五時，如何利用數值方法近似求解其根」。  
由於高次多項式在一般情況下不存在封閉形式的根公式，因此本研究結合理論數學與電腦實作，透過 Python 實現數值求根演算法，驗證其可行性與準確度。

---

## 研究背景與動機
在中學數學中，我們熟悉一次、二次，甚至三次與四次多項式的根公式。然而，當多項式次數 \( n \ge 5 \) 時，根據 **:contentReference[oaicite:0]{index=0}** 所建立的伽羅瓦理論，一般情況下已不存在僅使用加減乘除與開根號表示的解析解。

儘管如此，高次多項式仍廣泛出現在：
- 控制系統的特徵方程式
- 數值微分方程的穩定性分析
- 工程與機器學習中的最佳化問題  

因此，數值方法成為實務上不可或缺的解決方案。

---

## 問題定義
給定一個多項式：

\[
f(x) = c_0 + c_1 x + c_2 x^2 + \cdots + c_n x^n
\]

目標是找出使得：

\[
f(x) \approx 0
\]

的實數或複數解。

本專題的解題流程如下：

1. **數學理論分析**
   - 說明高次多項式無法使用解析根公式的原因
   - 理解「根」在複數域中的意義

2. **問題轉換**
   - 將「解方程式」轉換為「最小化 \(|f(z)|\) 的問題」

3. **選擇數值方法**
   - 牛頓法（Newton’s Method）
   - 爬山法（Hill Climbing，搭配隨機重啟）

4. **程式實作**
   - 使用 Python 實作多項式計算與數值演算法
   - 支援複數運算

5. **驗證與分析**
   - 將近似根代回原多項式
   - 評估誤差與收斂情形

## 使用的數值方法

### 1. 牛頓法（Newton’s Method）
利用切線逼近根，迭代公式為：

\[
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
\]

**優點**
- 收斂速度快（平方收斂）

**缺點**
- 需要計算導數
- 初始值不佳可能導致發散

---

### 2. 爬山法（Hill Climbing）
將求根問題轉為最佳化問題：

\[
g(z) = |f(z)|
\]

透過在複數平面中隨機嘗試鄰近點，逐步減少 \(|f(z)|\)。

**特點**
- 可自然處理複數根
- 不需解析解或導數
- 容易陷入局部極小值（使用隨機重啟改善）
